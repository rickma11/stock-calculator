# 小红书爆款封面图生成器

## 功能概述

本 skill 用于自动生成小红书风格的爆款封面图，支持多种模板风格、文字排版优化和配色方案推荐。

## 核心功能

### 1. 自动生成封面图
- 根据文章标题和关键词自动生成封面图
- 智能提取关键信息进行排版
- 自动调整文字大小和位置

### 2. 多种模板风格
- **简约清新风**：干净简洁的设计风格
- **高饱和度吸睛风**：鲜艳色彩吸引眼球
- **渐变背景风**：使用渐变色背景
- **图文混排风**：文字和图片混合排版

### 3. 文字排版优化
- 自动优化标题文字的排版和字体
- 支持多行标题自动换行
- 智能调整字体大小和粗细

### 4. 配色方案推荐
- 根据内容推荐合适的配色方案
- 支持自定义配色
- 提供多种预设配色方案

### 5. Midjourney AI 生成（新功能）
- 支持 Midjourney API 集成
- 自动生成精美的 AI 图片
- 智能生成 Midjourney 提示词
- 支持本地生成和 AI 生成两种模式

## 使用方法

### 交互式引导（推荐）

使用交互式引导功能，通过选择题和填空题逐步完成封面图制作：

```bash
python interactive_generator.py
```

**交互流程**：
1. 输入文章标题
2. 输入关键词（可选）
3. 选择封面风格（4种可选）
4. 选择配色方案（10种可选）
5. 选择输出格式（JPG/PNG）
6. 选择生成方式（本地生成 / Midjourney）
7. 确认信息并生成

### Midjourney 配置

详细配置说明请查看 [MIDJOURNEY_SETUP.md](./MIDJOURNEY_SETUP.md)

**快速配置**：
```bash
# 设置环境变量
export MIDJOURNEY_API_KEY=your_api_key

# 或在交互式输入 API Key
python interactive_generator.py
```

## 输入参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| title | string | 是 | 文章标题 |
| keywords | array | 否 | 关键词列表 |
| style | string | 否 | 封面风格（默认：简约清新风） |
| color_scheme | string | 否 | 配色方案 |
| output_format | string | 否 | 输出格式（默认：JPG） |

## 输出结果

生成的小红书封面图，保存为 JPG 格式。

## 模板样式

### 简约清新风
- 背景色：浅色系（米白、浅灰、淡蓝）
- 文字颜色：深色系（深灰、深蓝）
- 字体：无衬线字体
- 排版：居中对齐，留白充足

### 高饱和度吸睛风
- 背景色：鲜艳色（红、橙、黄、粉）
- 文字颜色：白色或对比色
- 字体：粗体无衬线字体
- 排版：大标题，醒目突出

### 渐变背景风
- 背景色：渐变色（蓝紫渐变、橙红渐变）
- 文字颜色：白色
- 字体：现代感字体
- 排版：层次分明，视觉冲击力强

### 图文混排风
- 背景色：图片或渐变
- 文字颜色：根据背景调整
- 字体：多样化字体组合
- 排版：图文结合，创意排版

## 技术实现

使用 Python 的 Pillow 库进行图像处理，支持：
- 动态创建画布
- 文字渲染和排版
- 颜色渐变
- 图像合成
- 多格式输出

## 注意事项

1. 标题长度建议控制在 20 字以内
2. 关键词建议 3-5 个
3. 输出图片尺寸为 1080x1440（小红书标准尺寸）
4. 支持中英文混合排版
