## 使用技巧

### 1. 快速跳过

- 输入关键词时直接回车可以跳过
- 选择风格和配色时直接回车使用默认选项

- 完成一张封面图后，选择继续制作

## 常见问题

### Q: 标题太长怎么办？

A: 系统会提示标题过长，您可以选择继续或重新输入。建议缩短标题到 20 字以内。

### Q: 关键词必须输入吗？

A: 不是必须的。您可以直接回车跳过关键词输入。

### Q: 如何选择合适的风格？

A: 参考风格说明，根据内容类型选择：
- 知识类：简约清新风
- 推广类：高饱和度吸睛风
- 科技类：渐变背景风
- 品牌类：图文混排风

### Q: 生成的图片在哪里？

A: 图片保存在 `output` 目录中，文件名格式为 `{标题}_cover.jpg` 或 `{标题}_cover.png`。

### Q: 可以修改已生成的图片吗？

A: 目前不支持直接修改。建议重新运行生成器，调整参数后重新生成。

### Q: 支持哪些字体？

A: 系统会自动加载系统字体。Windows 系统使用微软雅黑字体。

## 高级用法

### 直接调用 Python API

```python
from generator import XiaohongshuCoverGenerator

generator = XiaohongshuCoverGenerator()

output_path = generator.generate(
    title="您的标题",
    keywords=["关键词1", "关键词2"],
    style="简约清新风",
    color_scheme="紫色渐变",
    output_format="JPG"
)

print(f"封面图已生成: {output_path}")
```

### 批量生成

```python
from generator import XiaohongshuCoverGenerator

generator = XiaohongshuCoverGenerator()

titles = [
    {"title": "标题1", "keywords": ["关键词1"]},
    {"title": "标题2", "keywords": ["关键词2"]},
    {"title": "标题3", "keywords": ["关键词3"]}
]

results = generator.generate_batch(titles)

for result in results:
    print(f"生成成功: {result}")
```
