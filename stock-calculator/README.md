# 股票买卖计算器

一个简单易用的股票买卖计算器网页应用。

## 功能特性

- 股票买卖价格计算
- 今日涨跌停价格计算
- 实时大宗商品价格跳转（油价、金价、银价、铜价）

## 文件说明

- `stock-calculator.html` - 股票计算器主页面
- `index.html` - 股票计算器页面（备用）
- `.gitignore` - Git 忽略配置

## 使用方法

直接在浏览器中打开 `stock-calculator.html` 或 `index.html` 即可使用。

## 部署

可以部署到 GitHub Pages 等静态网站托管服务。
