# 股票买卖计算器

一个简单易用的股票买卖成本价计算器，可以计算买卖后的成本价、盈亏情况和持仓市值。

## 功能特性

- 支持买入和卖出操作
- 实时计算新成本价
- 显示盈亏金额和盈亏比例
- 显示当前持仓市值
- 交易摘要记录

## 使用方法

1. 在浏览器中打开 `stock-calculator.html` 文件
2. 填写当前持仓信息（成本价、持有数量）
3. 选择交易类型（买入/卖出）
4. 填写交易价格和数量
5. 点击"计算"按钮查看结果

## 计算逻辑

**买入时**：
- 新成本价 = (原成本价 × 原数量 + 买入价 × 买入数量) ÷ (原数量 + 买入数量)

**卖出时**：
- 盈亏金额 = (卖出价 - 原成本价) × 卖出数量
- 盈亏比例 = (卖出价 - 原成本价) ÷ 原成本价 × 100%
